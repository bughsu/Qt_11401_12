# Qt6 專業多路監控錄影系統 - 智慧移動偵測 + 遠端監看

專業的多路視訊監控系統，整合 OpenCV 智慧移動偵測、完整的事件日誌功能和遠端 Web 監看。

## ⭐ 核心功能

### 遠端 Web 監看 🌐 (新功能)
- 內嵌輕量級 Web Server，無需額外軟體
- 即時影像轉碼為 MJPEG 串流透過 HTTP 輸出
- 支援手機、平板瀏覽器遠端觀看
- 自動偵測本機 IP，方便行動裝置連接
- 響應式網頁設計，適配各種螢幕尺寸

### 智慧移動偵測
- 即時影像分析，使用 OpenCV MOG2 背景減除算法
- 當畫面像素變化超過閾值時自動觸發事件
- 可調整靈敏度（1%-10%）
- 自動儲存移動偵測快照

### 事件日誌系統
- 完整記錄所有系統事件
- 移動偵測、錄影、串流連接等事件追蹤
- JSON 格式持久化儲存
- 事件瀏覽和管理介面

### 多路監控錄影
- 同時支援多個串流來源
- 支援 RTSP、HTTP、MJPEG 等格式
- 偵測到移動時自動錄影（可選）
- 使用 FFmpeg 高效錄影

## 📋 系統需求

- Qt 6.x (含 Network 模組)
- FFmpeg 8.0.1+
- OpenCV 4.x
- C++17 編譯器

## 🚀 快速開始

1. 安裝依賴項（Qt6、FFmpeg、OpenCV）
2. 設定 `Topic.pro` 中的路徑
3. 使用 Qt Creator 建置專案
4. 執行程式並新增串流來源
5. 啟用移動偵測功能
6. 啟用遠端監看，用手機瀏覽器連接

## 📸 主要介面

- **監控畫面**：九宮格多路顯示
- **移動偵測控制**：靈敏度調整、自動錄影選項
- **遠端監看控制**：啟用/停用 Web Server，顯示連接網址
- **事件日誌**：查看所有記錄的事件
- **檔案管理**：內建播放器和檔案管理

## 🔧 技術架構

- **前端**：Qt6 Widgets
- **視訊處理**：Qt6 Multimedia + FFmpeg
- **影像分析**：OpenCV 4.x
- **Web 串流**：Qt6 Network (QTcpServer) + MJPEG
- **資料儲存**：JSON 檔案格式
